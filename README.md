# College Management System

A REST API built with Python Flask and MongoDB for managing a college's core entities.

## Tech Stack
- **Backend:** Python + Flask
- **Database:** MongoDB Atlas
- **API Testing:** Postman

## Entities
- Departments
- Staff
- Students
- Books

## Features
- Full CRUD operations on all entities
- One-to-Many relationships (Department → Staff, Students, Books)
- Many-to-One relationships (Staff/Students → Department)
- REST API tested via Postman
